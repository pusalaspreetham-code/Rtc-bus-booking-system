# RTC Bus Booking System

A C++ console-based program for generating RTC bus tickets, calculating fare, saving ticket details to files, and printing the daily waybill summary.

---

## Requirements
- GCC / g++ compiler
- Windows / Linux / MacOS (any OS supporting GCC)
- No external libraries required

---

## How to Compile (Using GCC/g++)
Open the terminal in the folder where `project.cpp` is located, then run:

g++ project.cpp -o project

---

## How to Run
./project

---

## Project Files
- project.cpp      → Main source code
- tickets.txt      → Auto-generated ticket records
- waybill.txt      → Auto-generated daily summary
- readme.md        → Instructions to compile and run the project

---

## Notes
- `tickets.txt` and `waybill.txt` will be created automatically when the program runs for the first time.
- Make sure all files are in the same folder while running the program.
